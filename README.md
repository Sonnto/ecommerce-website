An e-commerce website inspired by the designs of Amazon. You can sign in through Google (Apple, GitHub, and Facebook is displayed but coming soon) purchase items through the store, and pay for it via Stripe.

- CSS

- [Stripe](https://stripe.com)

# Instructions

## Using npm

Run commands

1. `npm install`

2. `npm run dev`

## Or using yarn

Run commands

1. `npm install --global yarn`

2. `yarn install`

3. `yarn run dev`

## What to do?

Click to navigate the website. Add, remove, purchase items. Use credit card 4242 4242 4242 for test.
